Add or Edit Schedule Entry Help

You can create an entry for the schedule list on this page.

  • Description - In this field, create a description for the schedule. Note that this description should be unique, for example Schedule_1.
  • Apply To - In this field, select "Each Week" and day (days) you need, or choose "Each Day".
  • Start/End Time - In these field, you can select the Start Time and End Time in the corresponding field.

Click Save to complete the settings.

Click Back to return to the Schedule Settings page.
