Yandex.DNS

On this page you can configure Yandex.DNS filter which will secure your devices from malicious websites and will restrict adult websites for your children devices. You can configure a general filter for all devices or a separate filter for each device.

Please visit dns.yandex.ru to know more about Yandex.DNS service.

  • Enable Yandex DNS for all devices - This Yandex.DNS filter will be enabled on all devices, which does not exist in Yandex.DNS rule list.
  • Disabled - Means, that no any Yandex.DNS filter will work on all devices.
  • Basic - Means, that Basic Yandex.DNS filter will work on all devices.
  • Safe - Means, that Safe Yandex.DNS filter will work on all devices.
  • Child - Means, that Child Yandex.DNS filter will work on all devices.
  • MAC Address - Display deviceâ€™s MAC address on which Yandex.DNS filter will work.
  • Client Name - Display device description for better device identification.
  • Control Mode - Disaply chosen Yandex.DNS filter for special device.
  • Edit - To edit or delete an existing entry.

To configure specific Yandex.DNS filter for special device click Add. You can chose device from Scan list or enter MAC address manually. Select special control mode (basic, safe, child) and give a little description to rule.
