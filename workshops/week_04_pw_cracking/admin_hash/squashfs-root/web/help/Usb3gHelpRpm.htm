3G/4G Settings

  • Enable 3G as a backup solution for Internet access - Select the checkbox to enable 3G/4G as the backup solution for Internet access.
  • 3G/4G USB Modem - Displays whether or not a 3G/4G USB modem is plugged into the router.
  • PIN Status - Displays the PIN status of the SIM card, including Unable to unlock SIM card, Ready, SIM Locked, PIN Locked, and Unknown Error.
  • Location - Displays the location of the SIM card.
  • Mobile ISP - Displays the ISP (Internet Service Provider) of the 3G or 4G network. You can select your mobile ISP after you enable 3G/4G as the backup solution for Internet access.
  • Set Dial Number, APN, Username and Password manually - Select the checkbox to set the Dial Number, APN, Username and Password manually.
  • Dial Number/APN/Username/Password - Enter the Dial Number, APN, Username and Password provided by your ISP.
  • Authentication Type - Select an authentication type from the drop-down list. The default method is AUTO_AUTH. Some ISPs may require a specific authentication type, please confirm it with your ISP or keep the default settings.
  • MTU Size (in bytes) - The typical MTU (Maximum Transmission Unit) size for 3G or 4G network is 1480 Bytes.
  • Echo Request Interval - Enter a time interval value between 0 and 120 (in seconds) for which the router requests Access Concentrator to echo at every interval. The default value is 30. 0 means no detection.
  • Use The Following IP Address - Select this option and enter the IP address provided by your ISP.
  • Use The Following DNS Servers - Select this checkbox and enter the DNS server address(es) in dotted decimal notation provided by your ISP. This 3G/4G connection will only use the specified DNS server(s).
  • Use The Following IP Address - Select this option and enter the IP address provided by your ISP.
